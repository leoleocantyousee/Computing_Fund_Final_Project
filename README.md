# LKR Library Management System

LKR Library Management System is a Python application simulating a simple library.  
It allows users to acsess features such as registration and login, allows users to view available books and their details. On top of all
of this users can check books in and out.

The system keeps track of each user's checked out books and stores them using text files.  
On top of all that it generates an HTML file which can be launhed using any broweser creating a simple web interface that allows users to interact  
with the library catalog through their browser.
